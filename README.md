# سيرة عمر بن الخطاب رضي الله عنه

موقع ويب تفاعلي عن سيرة الخليفة الثاني عمر بن الخطاب رضي الله عنه.

## المميزات
- تصميم عصري متجاوب
- وضع قراءة ليلي/نهاري
- شريط تقدم القراءة
- أقسام تفاعلية
- أكورديون للقراءة العميقة
- دعم كامل للغة العربية

## التقنيات المستخدمة
- HTML5
- CSS3 (مع Tailwind CSS)
- JavaScript
- Google Fonts (Cairo)

## كيفية التشغيل
1. افتح `index.html` في المتصفح
2. أو استخدم Live Server في VS Code
